# Prefix Tree

## 설명
- 문자열 검색을 하는데 유용한 Tree이다.
- 예를 들어) ["go", "gone", "guild"]라는 input이 있을 때 prefix tree를 만들게 되면 아래와 같은 그림으로 만들어진다.<br><br>
